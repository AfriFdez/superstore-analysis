# Análisis de Superstore

Este proyecto incluye análisis estadístico y visualizaciones, así como un modelo de regresión.

## Contenido

- Análisis estadístico de las ventas y ganancias.
- Visualización de datos utilizando gráficos.
- Modelo de regresión para predecir ventas.

## Requisitos

- R
- RStudio
- Paquetes: tidyverse, lubridate, caret, scales

## Cómo Ejecutar

1. Clona el repositorio.
2. Abre el proyecto en RStudio.
3. Ejecuta el script principal.

## Resultados

Los resultados incluyen gráficos y un modelo de regresión para predecir ventas.
